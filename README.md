## Usage
```
from pymathutils import factorial, fibonacci, is_prime, gcd, lcm

print(factorial(5))   # 120
print(fibonacci(7))   # 13
print(is_prime(29))   # True
print(gcd(12, 18))    # 6
print(lcm(12, 18))    # 36
```
